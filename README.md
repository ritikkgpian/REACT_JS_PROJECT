<h4>A single page  application built with React.js which provides a comprehensive 
platform for campus users to get updated about the events happening in the campus and 
any notice related to thier Academics and Placements.</h4>

<h4>Tech stacks used:</h4>

<h4>HTML,Vanilla CSS,Javascript,React JS,React Router, Redux Toolkit</h4>

<h4>Key Features:</h4>

<h4>1)A welcome statement and visual appealing campus imagery. At the navbar section
,there is a clear navigation to all the sections of the application.</h4>

<h4>2)In the Notice and Events route, advanced search and filtering  by category capabilities are implemented. After search and category selecting,a browse feature button dynamically 
displays relevant notice or event in the form of cards.
</h4>

<h4>3)For a detailed view of a particular notice or event,there is a 
view button,accessible via specific URLs(ex: /notices/:id or /events/:id)
</h4>

<h4>4)In a particular card, there is a save button which allows users to bookmark important
notices and events along with real-time toast notifications whether it is saved or not </h4>

<h4>5)In the save route,all the details are shown along with date and time of SAVE and easy removal of items at any time with a remove button</h4>

<h4>6)A dedicated Login page for user Sign-In and form handling and form validation techniques are also implemented </h4>

<h4>7)During the fetching of data, loading indicators are displayed and when 
no such notices or events are present ,a friendly notice is also displayed.</h4>
